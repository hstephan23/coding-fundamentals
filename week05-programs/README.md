# Week 05 - Arrays & Collections

## Learning Goals 

- Work with many values
- Learn indexing patterns

### Mental Model 

Arrays are rows of boxes.

#### Prompts

- How many elements exist here? 

    5 elements

- What does index 0 represent? 

    The first element of the array

- What happens if I go out of bounds? 

    The program can crash, or you can get garbage which can cause major issues going forward
